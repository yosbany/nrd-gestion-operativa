# NRD Pedidos

Sistema de gestión de pedidos - PWA

## Características

- Gestión de clientes
- Gestión de productos
- Creación de pedidos
- Envío de resumen por WhatsApp
- Impresión de pedidos
- Funciona offline (PWA)

## Tecnologías

- HTML, CSS, JavaScript (Vanilla)
- Firebase Realtime Database
- Firebase Authentication
- Service Worker (PWA)
